Welcome to the PowerFAIDS™ Resource Center!

In observance of Independence Day, the College Board and PowerFAIDS Support will be closing at 3:00 p.m. Eastern Wednesday, July 3, and will remain closed Thursday, July 4.

We will resume our regular customer support schedule on Friday, July 5 at 9:00 a.m. Eastern.

What's New

  • An updated version of PowerFAIDS 29.3 is now available for download on our SFTP site. You must be in production with any version of PowerFAIDS 29.2 before you can upgrade to 29.3. Schools still working in any version of PowerFAIDS 29.2 are able to update directly into the most updated version available of PowerFAIDS 29.3 General Release.
  • We encourage you to regularly check Known Issues as we continue to identify discrepancies with production ISIRs and processes. 
  • As a reminder, support of PowerFAIDS Desktop will end with the 31.X release and the 2026-27 Award Year. All institutions will be required to migrate to PowerFAIDS Cloud to support processing for the 2027-28 Award Year in calendar year 2026.

The PowerFAIDS User Conference returns in person to Atlanta, Georgia on July 22-23, 2024!
Visit the User Conference website to find information about registration, agenda, and more.

Technology Announcements

  • Windows Server 2022 is supported beginning with the PowerFAIDS 28.2 release.
  • Windows 11 is supported beginning with the PowerFAIDS 28.2 release.
  • Microsoft Office 2019 is supported beginning with the PowerFAIDS 28.2 release.
  • Microsoft SQL Server 2019 will be supported through the end-of-life of PowerFAIDS Desktop.
  • PowerFAIDS Support for Internet Explorer 8 and earlier will end with the 29.3 release.  Microsoft ended support on October 13, 2020.

Products No Longer Supported by PowerFAIDS

  • PowerFAIDS Support for SQL Server 2017 ended with the PowerFAIDS 29.2 release. Microsoft ended mainstream support on October 11, 2022.
  • PowerFAIDS Support for SQL Server 2016 ended with the PowerFAIDS 26.2 release. Microsoft ended mainstream support on July 13, 2021.
  • PowerFAIDS Support for SQL Server 2014 ended with the PowerFAIDS 26.2 release. Microsoft ended mainstream support on July 9, 2019.
  • PowerFAIDS Support for SQL Server 2012 ended with the PowerFAIDS 26.2 release. Microsoft ended mainstream support on July 11, 2017.
  • PowerFAIDS Support for Microsoft Office 2013 ended with the 28.3 release.  Microsoft will end support on April 11, 2023.
  • PowerFAIDS Support for Microsoft Office 2010 ended with the 26.1 release.  Microsoft ended support on October 13, 2020
  • PowerFAIDS Support for Windows 7, Windows Server 2008, and Windows Server 2008 R2 ended with the 25.2 release. Microsoft support ended on January 14, 2020
  • Microsoft has disabled Transport Layer Security (TLS) versions 1.0 and 1.1
